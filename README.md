# vue-base-converter

This is simple app to convert numbers from one numeral system to another. Built with Vue.js, Typescript and Buefy.

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Run your unit tests

```
yarn test:unit
```

### Lints and fixes files

```
yarn lint
```
